# 📈 AI 주식 통합 진단 대시보드

Google Gemini AI를 활용한 개인 투자자용 주식 분석 대시보드입니다.

## ✨ 주요 기능

### 1. 재무 지표 자동 분석
- PER, PBR, ROE, 배당수익률 등 8가지 핵심 지표
- 시가총액, 52주 최고/최저가, 베타 값 제공

### 2. 눌림목(Pullback) 패턴 탐지 🎯
자동으로 다음 3가지 조건을 검증합니다:
- ✅ 주가가 20일 이동평균선 상단에 위치
- ✅ 20일선과의 거리가 3% 이내로 근접
- ✅ 최근 거래량이 감소 (에너지 응축 신호)

### 3. 기술적 지표
- 이동평균선: 5일, 20일, 60일, 120일
- RSI (14일): 과매수/과매도 판단
- MACD: 추세 전환 신호
- 거래대금: 시장 관심도

### 4. 거시 경제 지표 연동 🌍
- 미국 10년물 국채 금리 (^TNX)
- 원/달러 환율 (KRW=X)
- 전일 대비 변화율 자동 계산

### 5. AI 종합 분석 리포트 🤖
Google Gemini AI가 다음을 제공합니다:
- 매수/매도/보유 투자 의견
- 기술적/재무적 근거
- 거시 환경 영향도 분석
- 위험 요소 및 주의사항
- 목표가 및 손절가 제시

## 🚀 설치 방법

### 1. 파이썬 설치
Python 3.8 이상이 필요합니다.

### 2. 라이브러리 설치
```bash
pip install -r requirements.txt
```

### 3. Google AI Studio API 키 발급 (무료) 🆓
1. https://aistudio.google.com/app/apikey 접속
2. Google 계정으로 로그인
3. **"Create API Key"** 버튼 클릭
4. 생성된 키 복사 (예: `AIzaSy...`)

> ✅ **완전 무료**로 사용 가능합니다!  
> 일일 할당량: 1,500 requests/day (개인 사용 충분)

### 4. API 키 설정 (자동 입력)

**방법 1: secrets.toml 파일 사용 (권장)**
```bash
# 프로젝트 폴더에 .streamlit 디렉토리 생성
mkdir .streamlit

# secrets.toml 파일 생성 및 편집
nano .streamlit/secrets.toml
```

`.streamlit/secrets.toml` 파일 내용:
```toml
GOOGLE_API_KEY = "여기에_발급받은_API_키_입력"
```

**방법 2: 코드에 직접 입력**
`main.py` 파일의 104번째 줄을 수정:
```python
default_api_key = st.secrets.get("GOOGLE_API_KEY", "여기에_발급받은_API_키_입력")
```

> ⚠️ **보안 주의**: secrets.toml 파일은 절대 Git에 커밋하지 마세요!  
> `.gitignore` 파일에 이미 추가되어 있습니다.

## 💻 실행 방법

```bash
streamlit run main.py
```

브라우저가 자동으로 열리며 http://localhost:8501 주소로 접속됩니다.

## 📖 사용 가이드

### 1단계: 시장 선택
- **US**: 미국 주식 (NASDAQ, NYSE 등)
- **KR**: 한국 주식 (KOSPI, KOSDAQ)

### 2단계: 종목 코드 입력

**미국 주식 예시:**
- AAPL (애플)
- TSLA (테슬라)
- NVDA (엔비디아)
- MSFT (마이크로소프트)

**한국 주식 예시:**
- 005930 (삼성전자)
- 035720 (카카오)
- 000660 (SK하이닉스)
- 051910 (LG화학)

### 3단계: API 키 입력
왼쪽 사이드바에 Google AI Studio API 키를 입력합니다.

### 4단계: 분석 실행
**"📊 분석 실행"** 버튼을 클릭하면:
1. 재무 지표가 상단에 표시
2. 캔들스틱 차트와 이동평균선 렌더링
3. 눌림목 패턴 자동 탐지
4. 거시 경제 지표 표시
5. AI가 종합 분석 리포트 생성

## 🔍 눌림목 판별 로직 상세

```python
# 조건 1: 20일선 상단 위치
현재가 > 20일 이동평균선

# 조건 2: 20일선 근접 (3% 이내)
(현재가 - 20일선) / 20일선 * 100 ≤ 3%

# 조건 3: 거래량 감소 (에너지 응축)
최근 5일 평균 거래량 < 이전 10일 평균 거래량
```

**✓ 3가지 조건 모두 만족** → 눌림목 패턴 감지!

## ⚠️ 주의사항

1. **투자 판단의 참고 자료**일 뿐이며, 최종 투자 결정은 본인의 책임입니다.
2. AI 분석은 과거 데이터 기반이므로 미래 수익을 보장하지 않습니다.
3. 실시간 데이터가 아닌 **15-20분 지연 데이터**를 사용합니다.
4. 한국 주식의 경우 재무 지표가 제한적일 수 있습니다.
